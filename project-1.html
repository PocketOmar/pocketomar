Spotify-Osint

An open Source Program which allows the user who uses it to scrap spotify user information using spotify' official API

Project Overview

Spotify-Osint is a simple program which requires a users developer client information to login and scrape public user data such as total followers, display names, number of playlists and etc, it can also write information to txt upon cli request
